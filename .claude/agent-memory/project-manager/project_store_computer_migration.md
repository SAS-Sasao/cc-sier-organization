案件概要: コンビニエンスストアのストアコンピューター（ストコン）をオンプレミスからAWSクラウドへ移行し、機能刷新も行うプロジェクト。笹尾豊樹が2026年6月から「特定機能を実装するチームのPM」として参画予定。

**Why:** 現行システムのリプレースと機能刷新の両立が求められる大規模案件。SIer多社協業での機能チームPM役割。

**How to apply:** PM支援タスクでは、機能チームPM視点（依存関係管理・ステークホルダー調整・カットオーバー計画）を優先的に取り上げること。
