## 調査内容の概要

### ストアコンピューター (2026-03-21)

コンビニのストコンについて以下を網羅的に整理した。

- ストコンの定義・役割・1974年〜2020年代の歴史的変遷
- 主要チェーン（セブンイレブン/ファミリーマート/ローソン等）のシステム事情
- 機能詳細: 発注（自動/推奨/手動）・廃棄管理・日次精算・在庫管理・棚卸・単品管理・検品・フェイスアップ・収納代行・従業員管理・本部通信
- システム構成: POS連携・EOB/EOS通信・マルチメディア端末・電子マネー・DC連携・温度管理
- 業務サイクル: 1日の業務フロー・発注から廃棄サイクル・双方向データフロー
- 用語集: 50語以上（5カテゴリ）
- クラウド移行時の考慮事項: 移行優先度・非機能要件（99.99%可用性・PCI DSS・食品衛生法）

**Why:** 2026年6月参画のクラウド移行案件向けの事前学習資料として作成。
**How to apply:** 今後のコンビニ関連調査・提案時に本ドキュメントをベースとして参照・更新すること。
