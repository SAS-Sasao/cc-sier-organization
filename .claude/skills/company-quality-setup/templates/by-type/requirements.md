# 要件定義書 品質チェックリスト

## 必須セクション

- [ ] プロジェクト概要・背景
- [ ] 目的・ゴール
- [ ] スコープ（対象範囲と対象外範囲）
- [ ] ステークホルダー一覧
- [ ] 機能要件一覧（番号付き）
- [ ] 非機能要件（性能・セキュリティ・可用性）

## 品質チェック

- [ ] 各機能要件にIDが振られている（例: FR-001）
- [ ] 非機能要件に数値目標が含まれている
- [ ] TBD・未定が残っていない（または課題として明示）
- [ ] 顧客確認事項・前提条件が明記されている
