## 2. BluStellar — 入社後に最も関わる戦略ブランド

### BluStellarとは
NECが「従来のSI型ビジネス」→「価値創造型ビジネス」へ転換するための中核ブランド。
「AIとデータを使って、顧客も気づいていない課題を解決する」がコンセプト。

### 主要な動き

1. **BluStellar 2（2026年度発表予定）**
   - 現在の約500種類の商材を「150セットのオファリング × 30セットのシナリオ」に体系化
   - 個社データ＋特化型Agentによるプロセス変革 → 複数Agent連携によるビジネス変革へ進化

2. **グループ全社展開（2026年4月1日〜）**
   - NECネッツエスアイ、NECネクサソリューションズ、NECフィールディングにBluStellar展開
   - DXの構想策定〜実装〜運用・保守をEnd to Endで提供する体制を確立

3. **「クライアントゼロ」アプローチ**
   - 顧客に売る前に自社の工場・サプライチェーンでAIを徹底運用
   - 失敗も成功もデータ化し、現場に即したDX提案に活かす

## 3. NEC製品 — リテール配属PMが知っておくべきもの

### 映像分析基盤（EVA）
- 複数カメラの映像から人の位置・動きを詳細分析
- **ヒートマップ、立ち寄り率**として可視化
- 防犯・マーケティング・おもてなし・業務効率化など複数用途に対応
- 同一プラットフォーム上に多様な映像分析機能を組み合わせ可能

### 人物行動分析サービス（2017年〜）
- 店舗カメラから来店者の**性別・年代・移動履歴・滞留状況**を分析
- **時間帯ごとの来店者数推移**、**滞留ヒートマップ**を可視化
- 利用料: 1店舗月額22万円〜
- 技術: NEC独自の「性別・年齢自動推定技術」「人物動線抽出技術」

### 自社LLM「cotomi」
- 高い日本語能力、金融・法律・製造など業界特化
- NEC Generative AI Service として企業向けに提供
- RAG対応を前面に打ち出し

### NeoSarf/POS（リテール向け）
- 有人レジ/フルセルフ/セミセルフの3形態対応

### storeGATE/L
- 本部・店舗一体型クラウドシステム（販売・仕入・在庫・発注を統合）

### 顔認証技術
- NISTベンチマーク世界1位
- 大阪・関西万博の多数の店舗で採用（2025年4月〜）

### セキュリティ
- CISSP資格保有者: 560人以上

## 4. AI戦略

- 2026年のトレンド: 生成AI → **Agentic AI**（自律的にタスクを実行するAI）
- 業界特化型AI（SME-LLM）の台頭
- 経営管理システムでの試験実装: 集計〜分析のリードタイム削減率 **96.7%**
- 定型判断業務をAIで約40%削減、PoC検証工数を約50%削減の見込み

## 5. 人材戦略・社内体制

### ジョブ型人材マネジメント
- 2024年4月: ジョブグレード体系と報酬制度を刷新
- 2025年4月: グループ6社（約49,000人）に共通基盤を展開
- パフォーマンスに基づくフェアな評価とフィードバックを徹底
- CHRO: 堀川大介氏（執行役 Corporate EVP 兼 CHRO）

### 等級と年収レンジ
```
C2（担当）      : 350〜600万
D1〜D2（主任）   : 700〜950万
M4（マネージャー）: 900〜1,100万+
M3（ディレクター）: 1,200万〜
```

### DX人材育成
- グループ全体で **12,000名** のDX人材育成目標
- 「BluStellar Academy」で540社以上に知見提供

### 組織再編（2025年4月1日付）
- クロスインダストリーBU・グローバルイノベーションBUの行政・医療事業 → パブリックBUに集約

### 社内環境
- コミュニケーション: Microsoft Teams中心
- テレワーク率高め（出社率40〜50%）
- フレックスタイム制・裁量労働制
- 1on1面談の全社推進

### 株式報酬制度「NECバリュー・シェアズ」
- 2025年度〜: 統括部長ら約400人を対象に開始
- 将来的にグループ6,000人以上に拡大予定
