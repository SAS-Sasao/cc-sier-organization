## 調査結果

### 1. WAKONXとは何か

**WAKONX（ワコンクロス）** は、KDDIが2024年5月に始動させた「AI時代のビジネスプラットフォーム」。名称は「和魂洋才（わこんようさい）」に由来し、日本の強み（おもてなし、現場力）と西洋の先端技術を融合する理念を体現。「X（クロス）」はあらゆるものを掛け合わせて無限の可能性を生むことを意味する。

**ポジショニング**: KDDIが「通信キャリア」から「社会課題解決プラットフォーマー」へ変革するための中核ブランド。NTTの「IOWN」、ソフトバンクの「AIコンピューティング」に対するKDDIの回答。

### 2. アーキテクチャ（3層構造）

| レイヤー | 機能 | 具体的な構成要素 |
|---------|------|----------------|
| **Network Layer** | 業界ごとに最適化された通信基盤 | 5G SA、固定回線、IoT、Starlink衛星通信、24/365運用監視 |
| **Data Layer** | 企業間データのセキュアな蓄積・融合・分析 | 顧客基盤（Ponta + au）、位置情報、購買データ、AIモデル開発環境 |
| **Vertical Layer** | 業界別にファインチューニングされたAI・ソフトウェア | 業界特化ソリューション群（後述の6業界プラットフォーム） |

### 5. ローソンでの具体的な活用事例・導入状況

#### 5-1. Real x Tech LAWSON（2025年6月〜）

高輪ゲートウェイシティに1号店オープン。15のリテールテック機能を搭載:

**顧客体験系**:
- **AIサイネージ**: AIカメラで顧客行動を解析し、関連商品をリアルタイム推奨。バナー広告比較で**約4倍以上の閲覧効果**
- **AI Ponta**: 生成AIロボットが来店者にレコメンド・周辺情報提供
- **プライスレール連動サイネージ**: 惣菜調理完了時にサイネージで一斉案内
- **3Dアバター遠隔接客**: Pontaよろず相談所で健康・金融相談を遠隔対応
- **スマートシティ連携**: 都市OSから天気・電車遅延・混雑情報をリアルタイム表示

**店舗運営系**:
- **自動調理ロボ**: からあげクン等の調理を自動化
- **飲料陳列ロボ**: ペットボトル等の棚補充を自動化
- **自動掃除ロボ**: 店内清掃の自動化
- **スマートシェルフ**: デジタル在庫棚でリアルタイム在庫管理
- **AIエージェント分析**: 防犯カメラ映像から業務量を算出、課題抽出〜改善提案

#### 5-2. ローソン S KDDI高輪本社店（2025年11月〜）

- VLA AIモデルによる自律ロボットが品出し作業を自動化
- 欠品検知から品出しまでの一連の流れを自動化する実証

#### 5-3. AIグラス実証（2025年10月〜12月）

- 従業員がAIグラス着用し業務撮影
- AIが作業内容・時間を分析・可視化
- 食品調理マニュアルをAIに取り込み、音声で手順確認可能
- 例: ご飯の盛り方をAIが評価し、偏りがあれば正しい盛り方を助言
- 今後、物流・製造分野への応用も検討

#### 5-4. KDDI Retail Data Consulting

- 位置情報データ + サービス利用状況 + 店舗売上・会員情報を統合ダッシュボード化
- 提供パック: 店舗開発支援、売上予測支援など5種類
- 2025年2月: 生成AIが出店候補地・プロモーション候補地を提案する機能追加

### 6. 顧客データ基盤の規模

| 指標 | 規模 |
|------|------|
| au PAY + Ponta会員 | **1億超** |
| auスマートパスプレミアム会員 | 1,500万人 → **2,000万人目標** |
| モバイル口座数 | **2,200万超** |
| 年間ポイント付与額 | **2,000億円超** |
| ローソン年間来店客数 | **約41億人** |
| ローソン店舗数 | **約14,600店舗** |

**国内最大級の顧客データ基盤**: au会員の位置情報・通信利用パターン + Pontaの購買履歴 + ローソンの来店データを統合。これにより「どこに住む、どんな属性の人が、何時に、何を買うか」を高精度に予測可能。

### 7. NECの映像AI/顔認証とWAKONXの連携可能性

#### 7-1. 既にあるKDDI x NEC パートナーシップ

- **サイバーセキュリティ合弁「United Cyber Force（UCF）」**: 2025年11月設立。能動的サイバー防御（ACD）で政府・法人向けソリューション提供
- **基地局・ネットワーク**: NECはKDDI向け5G基地局の主要サプライヤー

#### 7-2. 技術的な連携シナリオ

| 連携シナリオ | NEC側 | KDDI/WAKONX側 | 実現可能性 |
|-------------|-------|--------------|----------|
| 映像AI分析 + 5G伝送 | 映像認識AI（エッジ/クラウド） | 5G SA低遅延ネットワーク | 高（5G基地局の関係あり） |
| 顔認証 + Pontaデータ | NEC顔認証エンジン | Ponta会員データベース | 中（プライバシー課題あり） |
| 店舗分析 + 顧客行動 | dotData（購買パターン分析） | 位置情報 + 通信データ | 中（データ共有の契約次第） |
| セキュリティ + 店舗DX | UCF（サイバー防御） | WAKONX基盤 | 高（合弁関係あり） |

#### 7-3. 現時点での課題

- Real x Tech LAWSON 1号店にNEC技術の明示的な採用はない → KDDIが独自技術・パートナーで構成
- ローソン店舗DXの主導権はKDDI（50%出資の親会社）にある
- NECはローソンに対して「ベンダー」の立場であり、KDDI-ローソン-三菱商事の「経営パートナー」とは構造が異なる

### 8. 通信 x リテールの具体的ユースケース

#### 実現済み / 実証中

1. **リアルタイム商品レコメンド**: AIカメラで棚前行動を解析 → サイネージで即座に関連商品を提案
2. **調理タイミング連動告知**: 惣菜調理完了 → サイネージで一斉案内（「最高のタイミングで提供」）
3. **ロボット配達**: KDDI新本社ではロボットが各フロアへローソン商品を配達、フロア別売上差分析
4. **位置情報ターゲティング**: スマホの位置・購買情報で店舗状況に応じた特典配信
5. **AIグラスによる作業支援**: 従業員が手を止めずにAIにマニュアル確認
6. **VLAロボットによる品出し自動化**: 欠品検知→品出しの一連の流れを自動化

#### 構想中 / 将来計画

1. **ローソンタウン構想**: ローソン店舗を中核とした地域コミュニティDX（買い物難民救済、再エネ、Starlink）
2. **デジタルツイン活用**: 店舗・街区のデジタルツインでイベント・防災シミュレーション
3. **フィジカルAI**: 大阪堺データセンターの計算リソースを活用した高度なロボット制御
4. **海外展開**: Real x Tech LAWSON の実証結果をもとに海外店舗へ展開

### 9. 競合比較

| 項目 | KDDI（WAKONX） | ソフトバンク | NTTデータ/NTT |
|------|---------------|-------------|--------------|
| **リテール戦略** | ローソン50%出資で「経営」から関与 | ベンダーとしてソリューション提供 | セブン&アイ等の大型SI案件 |
| **映像解析** | AIカメラ + AIグラス（自社実証） | STAION（汎用映像解析PF） | − |
| **データ基盤** | Ponta + au会員（1億超）| PayPayデータ、Yahoo!検索データ | − |
| **AI基盤** | ELYZA（日本語LLM）、OpenAI連携 | SB Intuitions（自社LLM）| tsuzumi（NTT独自LLM） |
| **DC投資** | 大阪堺DC（NVIDIA GB200） | 北海道苫小牧DC（大規模投資） | − |
| **ロボティクス** | VLAモデルロボット（品出し・調理） | − | − |
| **小売特化度** | 極めて高い（経営参画） | 中程度（汎用SaaS） | 中程度（SI型） |

**ソフトバンクSTAIONの特徴**:
- 2022年5月提供開始の映像解析プラットフォーム
- 来店者カウント、属性分析、滞在時間計測、デジタルサイネージ視聴時間計測
- エッジ処理（映像データをクラウドに上げずセキュアに解析）
- 「サキミル」（来客数予測）+ 「らくしふ」（シフト管理）の連携でシフト作成工数7割削減
- ただし、KDDIのようなコンビニ経営への直接参画はなし

**KDDIの圧倒的優位性**: ローソンの経営に直接関与しているため、「ベンダーとして提案→採用してもらう」ではなく「経営判断としてテクノロジーを導入する」立場にある。これは他社には真似できない構造的優位。

## PMとしての示唆

### NEC視点での戦略的ポジショニング

1. **KDDIはパートナーであり競合**: ローソン店舗DXではKDDIが主導権を持つが、NEC-KDDI間にはUCF合弁やネットワーク機器供給の関係がある。敵対ではなく「KDDI WAKONXエコシステム内でNEC技術をどう位置づけるか」を考えるべき
2. **NECの差別化ポイント**: 映像認識AI（世界トップクラスの精度）、顔認証（世界No.1）、dotData（特徴量自動抽出）はKDDIにはない技術。WAKONX Data Layerへの技術供給者としてのポジションを狙える
3. **「BluStellar x WAKONX」の連携提案**: NEC側のBluStellarシナリオとKDDI側のWAKONX Retailを組み合わせた統合提案ができれば、両社Win-Win

### 入社後のアクション

1. **WAKONX Retailの内部資料を入手する**: KDDIとの契約関係、技術仕様の詳細を社内で確認
2. **Real x Tech LAWSON の技術スタックを分析する**: NEC技術が含まれていない理由を確認し、次フェーズでの参入余地を探る
3. **KDDI側のPM/意思決定者とのリレーション構築**: UCF合弁の関係者経由でローソンDXチームへのアクセスを確保

## 未解明の論点

1. **WAKONXの具体的なAPI仕様・技術ドキュメント**: 法人向けで一般公開されていない。パートナー企業としてアクセス可能か確認が必要
2. **ローソンDXの予算配分**: KDDI投資分とローソン自社投資分の内訳。NEC向け予算の規模感
3. **Real x Tech LAWSONの定量KPI**: 来店数・客単価・オペレーション効率の改善実績（内部データ）
4. **WAKONX Retailのロードマップ**: 高輪実証後の全国展開スケジュールと技術選定プロセス
5. **Ponta x au データ統合のプライバシー対応**: 個人情報保護法改正への対応状況、NEC顔認証との連携における同意取得フロー
6. **OpenAI連携の具体内容**: KDDI-OpenAIパートナーシップがローソンDXにどう適用されるかの詳細
