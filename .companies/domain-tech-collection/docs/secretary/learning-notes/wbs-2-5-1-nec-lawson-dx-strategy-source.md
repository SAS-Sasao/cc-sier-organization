## 1. ローソンの経営体制変化

### KDDI + 三菱商事の共同経営
- 2024年9月: KDDI + 三菱商事の共同経営体制確立（議決権50%ずつ）
- 2024年7月: 上場廃止 → 短期的株価配慮から解放 → 中長期DX投資に振れる体制
- **「Real×Tech Lawson」への進化を本格化**

### 三社の役割分担

| プレイヤー | 役割 |
|-----------|------|
| **KDDI** | 共同経営、通信基盤、会員データ連携、「Real×Tech Lawson」推進、WAKONX |
| **三菱商事** | 共同経営、サプライチェーン・物流ノウハウ、産業DXプラットフォーム |
| **ローソンデジタルイノベーション（LDI）** | ローソン×シグマクシス共同出資のIT戦略子会社。内製化の中核 |
| **NEC** | 映像認識AI・LLM実証、顔認証技術、SI案件 |

## 2. ローソンのDX施策一覧

### AI.CO（AI Customized Order）— 次世代発注システム
- ローソン**全店導入済み**
- 各店舗の特性（立地・客層・売上パターン）に応じた**個店別発注**を実現
- 値引き推奨: 対象商品の選定、効果的な値引き額・タイミングをAIが提案
- 2024年3月から新アルゴリズムで全国展開
- **品揃え・発注・値引き・廃棄（食品ロス）の4課題に対応**

### カンパニー制
- 地域ごとの意思決定スピードを上げる組織改革
- 「全国一律の最適解」→「地域特性に合わせた判断」への転換
- **個店最適化の組織的受け皿**

### 店舗運営支援AI
- 天井カメラ**8〜12台**で顧客行動を追跡
- 「どの棚で立ち止まり、どの商品に手を伸ばしたか」のデータ収集
- 個店ごとの施策を提案

### セルフレジ
- **4,500店舗以上**に導入（大手コンビニで唯一のフルオート展開）

### アバター接客「AVACOM」
- 無人店舗運営を実現

### ウォークスルー決済
- 東急電鉄二子玉川駅内で実装済み

### KDDI連携の顧客データ基盤
- KDDI + ローソンの会員情報を統合 → **国内最大級の顧客データ基盤**

## 3. NEC × ローソン 案件詳細

### 映像認識AI × LLM 実証実験

- **発表**: 2025年10月30日プレスリリース
- **時期**: 2025年10月27日〜11月2日
- **場所**: 埼玉県の1店舗
- **内容**:
  1. 店舗カメラで従業員の作業を撮影
  2. 映像認識AIが100以上の作業項目を自動分類
  3. LLMが作業工程・所要時間を文章化
  4. 定量レポートを自動生成
- **目標**: **店舗作業30%削減**
- **従来の課題**: 担当者が直接店舗に立ち会い手作業で計測 → 膨大な負荷
- **プライバシー**: 個人特定しない設計、実験後にデータ削除

### NEC発注支援AI（参考: セブン-イレブン向け）
- 「説明できるAI」: 発注推奨の理由を人間が理解できる形で提示
- 分析要素: 販売実績・カレンダー・天候・近隣イベント・キャンペーン・TV CM
- 効果: 発注時間35%削減、欠品率27%改善

## 4. セブン-イレブン × NEC（競合比較用）

### 次世代店舗システム（超大型案件）
- 全国約**21,000店舗**の業務システムをフルクラウド化
- **国内コンビニ初のフルクラウド型業務システム**
- **Google Cloud基盤** + マイクロサービスアーキテクチャ
- 発注・商品管理・従業員管理を効率化・高度化
- NEC顔認証技術を導入（約40万人の店舗従業員がログイン・入退室に利用）
- ServiceNow（ITSM）でオペレーション管理
- Omnissa Workspace ONEでデバイス管理
- 2025年春から全店舗に順次導入、2026年2月までに全店稼働予定

**ローソン担当としての示唆**: セブンはNECと深い関係で先行投資が進んでいる。ローソン向けに同等以上の提案ができるかが、NEC社内でのポジション確立に影響する。

## 5. 競合・先行事例

### トライアルカンパニー（国内最先進事例）
- 福岡のスーパーセンターに**700台のAIカメラ**を設置
- 全国216店舗で約19,500台のスマートカートを運用（世界最大規模）
- 顧客トラッキング: 来店客の人数・属性・移動経路・棚での滞在時間を記録
- **棚割り最適化**: P&G Japanとの協業で、機械学習+GANで最適棚配置を自動生成
- 購買行動の発見: ビール=計画購買（滞在短い）、菓子=衝動買い（滞在長い）
- 動的広告: カメラで顧客属性を認識 → ディスプレイ広告を自動切り替え
- 在庫管理: カメラで棚の欠品を監視 → 完売前にスタッフ通知

### Amazon Go / Just Walk Out（撤退トレンド）
- カメラ+重量センサー+ディープラーニングで「レジなし決済」を実現
- **2025-2026年: 店舗閉鎖が進行中。技術をRFIDベースに転換**
- **教訓**: 「レジなし」は費用対効果が合わなかった。**「行動分析→売上向上」** の方向が正解

### Walmart IRL（Intelligent Retail Lab）
- 店舗内に無数のカメラ・センサーを設置
- リアルタイムで商品の動きをモニタリング
- 在庫管理・棚割り最適化に活用

## 6. NEC映像分析技術 × ローソン — 提案拡張の可能性

### 現状: 従業員作業分析（コスト削減）
### 拡張案: 顧客行動分析 → 売上向上

**NEC既存製品で実現可能:**
- 映像分析基盤（EVA）: ヒートマップ + 立ち寄り率
- 人物行動分析サービス: 性別・年代・移動履歴・滞留状況

**提案の3層構造:**
1. **Layer 1**: 客動線ヒートマップ（時間帯別）
2. **Layer 2**: 個店別ゴールデンゾーン × 棚割り推奨
3. **Layer 3**: データドリブン店舗レイアウト（立地パターン別）
