## 1. NECのBU構造とリテールの位置

### 1-1. 2025年4月再編後のBU一覧

2025年4月1日付で実施された組織再編後のBU構造は以下の通り。

| BU名 | 主な事業領域 |
|------|-------------|
| **エンタープライズBU** | 金融、製造、**リテール**、トランスポート・サービス |
| パブリックBU | 中央官庁、自治体、教育（旧クロスインダストリーBU・GIBUの行政・医療を集約） |
| エアロスペース・ナショナルセキュリティBU（ANS） | 防衛、宇宙、サイバーセキュリティ |
| テレコムサービスBU | 通信事業者向け、海洋システム（海底ケーブル） |
| デジタルプラットフォームサービスBU（新設） | BluStellar軸の製品・サービス事業 |
| デジタルデリバリーサービスBU（新設） | コンサルティング・SIサービスのデリバリー |
| グローバルイノベーションBU | 海外DX、デジタルID |

**重要な整理**: 2025年4月の再編でクロスインダストリーBUが廃止されパブリックBUに集約されたのは「行政・医療」事業であり、リテールは影響を受けていない。リテールは一貫して**エンタープライズBU**に属する。

### 1-2. エンタープライズBU内のリテール組織階層

```
エンタープライズBU
 ├── 金融ソリューション事業部門
 ├── 製造ソリューション事業部門
 ├── リテールソリューション事業部門  ← ここ
 │    ├── 第一リテールソリューション統括部（大手コンビニ担当）
 │    │    ├── 第一インテグレーショングループ
 │    │    ├── 第三インテグレーショングループ（大手コンビニのDX・システムソリューション）
 │    │    └── ...
 │    ├── 第二リテールソリューション統括部（スーパー・専門店・EC等）
 │    └── ...
 └── トランスポート・サービスソリューション事業部門
```

- 第一リテールソリューション統括部が大手コンビニ（セブン・ローソン等）を担当
- NECソリューションイノベータ（NESIC）にも「EA・リテールソリューション事業部門」がグループ会社として並走
- セブン次世代システムでは NEC + NECソリューションイノベータが共同で開発

### 1-3. 2025年4月再編で押さえるポイント

1. **デジタルプラットフォームサービスBUが2分割**: サービスBU（BluStellar製品）+ デリバリーBU（SI/コンサル）
2. **テレコムBUは整理縮小方向**: 採算性の低い事業を整理。利益率10%以上を目標に
3. **ANSに海底ケーブル事業を移管**: テレコムBUから移管

**PMとして知っておくべきポイント**
ストコン案件の担当組織は「第一リテールソリューション統括部」配下と推定される。組織再編が頻繁であることを念頭に置き、参画時点での担当部署・担当統括部長を早期に確認すること。BluStellarのデリバリーを担う「デジタルデリバリーサービスBU」との役割分担も入社後に把握しておくと社内調整がスムーズになる。

```
全社 Non-GAAP OP率 7.5%（2024年度実績）
  → 10.1%（2025年度見通し）
  → 15.0%（2030年度目標）
         ↑
  BluStellar OP率 20.0% が全社を引き上げる主エンジン
  ANS 売上 7,000億円（次期中計）が第二エンジン
```

**PMとして知っておくべきポイント**
全社Non-GAAP OP率15%という目標は、現状の約7.5%から2倍への引き上げを意味する。この野心的な目標を支えているのはBluStellar（ITサービス中核）の高利益率化であり、PMが担当するすべての案件が利益率改善のターゲットになっている。「案件は取れたが利益が出ない」という状況はそのまま中計未達リスクにつながることを常に意識すること。

## 3. エンタープライズBU → リテール部門への分解

### 3-1. エンタープライズ全体の規模

NECは国内ITサービスを業種別に開示している。エンタープライズ（金融+製造+リテール+トランスポート・サービス）の2025年度通期見通しは**約6,009億円**。

| 区分 | 売上収益（2025年度見通し） |
|------|---------|
| パブリック | 7,111億円 |
| **エンタープライズ** | **6,009億円** |
| 子会社他（アビーム含む） | 8,239億円 |

### 3-2. リテール部門の推定規模

BU別の売上は非開示だが、以下の材料から推定できる。

| 推定根拠 | 内容 |
|---------|------|
| エンタープライズ全体 | 約6,009億円 |
| 金融・製造の比重 | エンタープライズ内で金融・製造が大きいと考えられる |
| **リテール単体（推定）** | **1,000〜1,500億円規模** |

### 3-3. 業種別受注動向からリテールを読む

#### 2024年度 通期受注動向

| 業種 | 前年比 |
|------|--------|
| パブリック | 好調（自治体標準化、消防防災） |
| 金融 | -9% |
| 製造 | +9% |
| **流通・サービス** | **+5%** |

#### 2025年度Q1 受注動向

| 業種 | 前年比 | 補足 |
|------|--------|------|
| パブリック | +7% | 自治体標準化・消防防災が牽引 |
| 金融 | -7% | — |
| 製造 | -2% | — |
| **流通・サービス** | **-19%** | **前年の大型案件（セブン次世代システム等）の反動。大型案件除くと前年並みを維持** |

Q1の-19%は見た目ほど悪くない。前年がセブン次世代システム等の大型案件で上期偏重だった反動であり、2025年度は大型案件の受注が下期偏重を想定している。

### 3-4. 利益率目標の分解構造

```
全社 Non-GAAP OP率 15.0%（2030年目標）
  ↓
ITサービス OP率 12.9%（現在）→ 上昇方向
  ↓
BluStellar OP率 20.0%（次期中計目標）
  ↓
シナリオ型案件の利益率 = ベース比 +3〜5%
  ↓
プロジェクト粗利率 30〜40%（推定目標）
  ↓
PM: 「人月積み上げ」でなく「価値ベースの提案」で案件を取る
```

**PMとして知っておくべきポイント**
リテール部門の売上目標は非開示だが、2024年度の流通・サービス受注が+5%と好調であったことは、担当部門の予算達成プレッシャーが継続していることを意味する。Q1の-19%はリテール担当マネージャーが社内で説明責任を問われる数値でもある。参画時期の組織の雰囲気（予算達成への焦りや大型案件への期待感）を早期に掴み、ストコン案件がその中でどう位置付けられているかを把握すること。

## 4. リテール部門PMの日常への落とし込み

### 4-1. ジョブ型評価体系とPMの位置

NECは2024年4月に全社員へジョブ型人材マネジメントを適用した。PM（主任〜マネージャー相当）の年収レンジと評価構造を理解しておく。

| 等級 | 役職相当 | 年収レンジ（推定） |
|------|---------|-----------------|
| C2 | 担当 | 350〜600万 |
| C1 | 担当（上位） | 500〜700万 |
| D2 | 主任 | 700〜850万 |
| D1 | 主任（上位） | 800〜950万 |
| **M4** | **マネージャー（課長級）** | **900〜1,100万+** |
| M3 | ディレクター（部長級） | 1,200万〜 |

### 4-2. PMが評価される2軸と9ブロック評価

**横軸（業績評価）**: 期初に立てた業績目標の達成度と成果

**縦軸（行動評価）**: 5つのCode of Valuesの実践度

| Code of Values | 内容 | ストコン案件への翻訳 |
|----------------|------|-------------------|
| 視線は外向き、未来を見通すように | 顧客・市場を見ているか | ローソンのDX戦略・KDDI連携を理解した提案 |
| 思考はシンプル、戦略を示せるように | 複雑なものを整理・方向性を示せるか | 移行方針を明快に説明できる |
| 心は情熱的、自らやり遂げるように | 主体的にやり切るか | QCD管理を自律的に推進 |
| 行動はスピード、チャンスを逃さぬように | 素早く動くか | 課題を即座にエスカレーション |
| 組織はオープン、全員が成長できるように | チームへの貢献 | ローソン・NEC双方チームの連携促進 |

### 4-3. 中計目標とPMの業績評価KPIの対応

| 中計目標 | PMの業績評価で測定されるもの（推定） |
|---------|-------------------------------------|
| BluStellar売上1兆円 | 担当案件のBluStellarへの貢献度（シナリオ/オファリング比率） |
| **営業利益率20%** | **プロジェクト粗利率**（予算対実績。最重要） |
| シナリオ構成比63% | 従来SI型→価値提案型への転換実績 |
| 全社Non-GAAP OP率15% | 高収益案件の獲得・完遂 |

### 4-4. PMが日常的に意識すべきKPI（推定）

| KPI | 説明 | 目安値 |
|-----|------|--------|
| **プロジェクト粗利率** | 売上 - 人件費 - 外注費。最重要KPI | **30〜40%** |
| 工数効率（予算消化率） | 見積工数に対する実績工数の比率 | 100%以内 |
| 追加受注・継続受注 | Phase2以降の拡張案件獲得 | — |
| 顧客満足度 | ローソン側の評価・次案件につながるか | — |
| BluStellarシナリオ比率 | 案件がシナリオ型として計上されるか | 対象案件の100% |

### 4-5. 利益率向上の4つの柱とPMの行動変容

| 柱 | 会社としての施策 | PMとしての行動変容 |
|----|---------------|--------------------|
| シナリオ型ビジネスへのシフト | シナリオ/オファリング比率33%→63% | ストコン案件を「モダナイゼーション系シナリオ」としてフレーミングして提案書・報告書に記載 |
| バリューベースプライシング | 成果に対する費用体系 | 「何人で何ヶ月」でなく「何を達成したらいくら」で提案書を構成 |
| 再利用・横展開による効率化 | シナリオのモデル化・横展開 | ローソンでの成功施策を他コンビニ・小売への横展開視点で常に記録 |
| AI活用による生産性向上 | cotomi等LLM活用で工数削減 | AI駆動開発スキルの習得が直接的に利益率改善に貢献 |

**PMとして知っておくべきポイント**
BluStellar 20%利益率の正体は「シナリオ構成比を33%→63%に引き上げること」によって達成するものである。ストコン案件の粗利率目標は30〜40%が期待値と推定される。参画直後に担当マネージャーへ「この案件の予算粗利率目標はいくらですか？」と確認することが第一歩。工数管理が甘いと即座に利益率を食うため、週次でのEVM（Earned Value Management）的なモニタリングを習慣化すること。

## 5. ストコン案件の財務的インパクト

### 5-1. 案件規模のスケール感

| 参照指標 | 数値 | ストコン案件との比較 |
|---------|------|-------------------|
| エンタープライズBU年間売上（推定） | 6,009億円 | ストコン案件の売上規模は非公開 |
| リテール部門売上（推定） | 1,000〜1,500億円 | — |
| セブン次世代システム（同業参照） | 約500億円規模（推定）・3,000人体制 | ストコン案件はこれより大幅に小規模と想定 |
| 流通・サービス受注増（2024年度） | +5%（年間） | ストコン案件がこの+5%の一部を構成 |

ストコン案件の個別の売上金額は非公開だが、大手コンビニ向け基幹システムのクラウド移行として数十億円規模（移行設計・構築+運用保守の複数年）と推定される。リテール部門全体（1,000〜1,500億円）の中では1〜5%程度だが、「ローソン向け継続案件のアンカー」として位置付くため、単年の売上額以上の戦略的価値を持つ。

### 5-2. BluStellarシナリオとしての財務貢献

ストコン案件がBluStellarのモダナイゼーション系シナリオとして計上される場合、従来SI型と比べて以下の財務的優位が生まれる。

| 項目 | 従来SI型 | BluStellarシナリオ型 |
|------|---------|----------------------|
| 単価設定 | 人月×単価（工数ベース） | バリューベース（成果ベース） |
| 利益率 | ベースライン（推定25%前後） | ベース比+3〜5%（推定30%超） |
| 契約形態 | 単発（請負・準委任） | コンサル→SI→運用の連鎖（長期） |
| 収益の予測可能性 | 案件ごとにリセット | 長期契約でストック型 |
| NECの社内評価 | 通常 | シナリオ比率63%目標に貢献として高評価 |

### 5-3. 「自分の案件の財務価値」を数字で語るフレーム

```
[ストコン案件の財務的位置付け]

NEC リテール部門（推定1,000〜1,500億円/年）
  └── ローソン向け売上（複数案件の合計）
        └── ストコン移行（設計・構築フェーズ）
              ├── 単年売上：数十億円規模（推定）
              ├── 粗利率目標：30〜40%（BluStellarシナリオ型として計上）
              └── 戦略的価値：Phase2〜4（AI.CO連携・映像AI等）への拡張の起点
```

**数字で語る際の3点セット:**
1. **売上貢献**: 「本案件はNECリテール部門売上の○%に相当する。ローソン向け最大規模の案件の一つ。」
2. **利益率**: 「BluStellarシナリオ型として計上することで、従来SI型比+3〜5%の粗利率を目標とする。」
3. **継続価値**: 「移行後のPhase2〜4（AI.CO連携、映像AI等）を含めると、5年LTVは初期構築費の3〜5倍と推定。」

**PMとして知っておくべきポイント**
社内プレゼンや予算会議で「自分の案件がなぜ重要か」を語る際、定性的な言葉（「ローソンとの関係が大事」）より財務数値（「粗利率目標30%超、Phase2以降のLTV試算○億円」）で語る方が上位層への訴求力が高い。ストコン案件を「NEC中計のリテール部門目標達成に直結するシナリオ型案件」と位置付け、Phase2〜4の拡張ビジョンを含む5年LTV試算を参画後3ヶ月以内に作成しておくことを推奨する。

## 6. 中計達成のリスクとPMの役割

### 6-1. NEC全社レベルの中計未達リスク

| リスク要因 | 内容 | PMへの影響 |
|-----------|------|-----------|
| 流通・サービス受注の変動 | Q1 -19%のような大型案件反動 | 自分の案件が「次の大型案件」として期待される圧力 |
| シナリオ比率目標（33%→63%）の達成 | 全PM横断の案件タイプシフトが必要 | 案件をシナリオ型として登録・管理する手間が増える |
| POS端末事業譲渡（2025年8月予定） | ハード売り→ソフト・サービスへの完全シフト | リテールの提案がよりソフト・クラウド寄りになる |
| KDDI×ローソンのIT内製化（LDI強化） | ローソン側のIT予算がNECに流れなくなるリスク | スコープ明確化と「NECでなければ実現できない」価値の確立が必要 |
| セブン-Google Cloud事例との比較 | ローソンが「なぜAWSか」を問う可能性 | AWS選定根拠の論理武装が必要 |

### 6-2. PMのコントロール可能範囲とコントロール外の整理

| 分類 | 内容 |
|------|------|
| **コントロール可能（PM直接）** | プロジェクト粗利率管理、工数効率、QCD達成、BluStellarフレーミング精度、追加受注交渉 |
| **コントロール可能（間接）** | ローソン担当者との関係構築（次フェーズ提案の地ならし）、社内リソース確保交渉 |
| **コントロール外（把握のみ）** | KDDI-ローソン間のIT戦略方針、NEC全社のシナリオ比率目標、リテール部門の年間予算配分 |

### 6-3. PMが特に注意すべき利益率の落とし穴

| パターン | リスク |
|---------|--------|
| 外注費の膨張 | 設計フェーズでの要件追加が外注費に直撃。粗利率を一気に下げる |
| 要件変更のスコープクリープ | 変更管理プロセスがないと無償対応が積み重なる |
| 工数見積もりの甘さ | オンプレ→クラウド移行はデータ移行・疎通確認工程で見積もりが甘くなりやすい |
| シナリオ型登録の漏れ | BluStellarシナリオとして社内登録しないと、利益率優遇が適用されない可能性 |

**PMとして知っておくべきポイント**
ストコン案件のPMとしてコントロールできる最大の変数は**プロジェクト粗利率**と**次フェーズの追加受注**である。中計未達リスクの多くはPMのコントロール外だが、「自分の案件だけは粗利率目標を達成し、追加案件につなげる」ことが最もNEC中計に貢献できる行動である。特にオンプレ→クラウド移行はデータ移行工程での工数超過リスクが高いため、移行リハーサル工程の工数を見積もりに必ず含め、変更管理プロセスをキックオフ時に合意しておくこと。

## 8. まとめ

### 8-1. 本ノートで得た核心的な理解

1. **NECの利益率向上は「シナリオ型への構成比シフト」が主エンジン**
   BluStellar全体の営業利益率20%目標は、シナリオ/オファリング比率を33%から63%に引き上げることで達成するものである。全PMに「自分の案件をシナリオ型として定義する責任」が課されていると読める。

2. **PMに降りてくる財務目標はプロジェクト粗利率30〜40%**
   全社Non-GAAP OP率15%→BluStellar OP率20%→シナリオ型+3〜5%という滝を下ると、プロジェクトレベルでは粗利率30〜40%が期待値として降りてくる。この数字を参画初日から意識してモニタリングすること。

3. **リテール部門はNECの隠れた重要領域（推定1,000〜1,500億円）**
   独立セグメントとして開示されないが、受注動向（2024年度+5%）とコンビニ業界での事例の厚みから、エンタープライズBU内での存在感は大きい。ストコン案件はその中でローソン向け継続案件のアンカーとして位置付く。

4. **ジョブ型評価体系でPMは業績（粗利率・売上）とCode of Valuesの2軸で評価される**
   特に「視線は外向き」と「行動はスピード」の2つのCode of Valuesが、ストコン案件PMとして日常で意識すべき行動基準として重要。
