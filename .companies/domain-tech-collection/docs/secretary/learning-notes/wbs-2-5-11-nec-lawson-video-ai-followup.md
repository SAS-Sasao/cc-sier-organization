# NEC x ローソン 映像AI実証実験「その後」深堀り調査

調査日: 2026-04-09
調査者: tech-researcher (Agent)

---

## 調査結果

### 1. パイロット実証の概要（振り返り）

- **期間**: 2025年10月27日〜11月2日（7日間）
- **場所**: 埼玉県の1店舗
- **技術**: NEC独自の映像認識AI + LLM（大規模言語モデル）
- **手法**: 店舗設置カメラで従業員を撮影 → 映像認識AIが100以上の作業項目に自動分類 → LLMが所要時間を文字化しレポートを自動生成
- **目標**: 店舗作業30%削減に向けた基礎データ取得
- **プライバシー**: 個人特定利用なし、実験終了後に映像データ削除

### 2. パイロット後の結果・成果報告

**2026年4月9日時点で、パイロット実証の定量的な結果報告は公開されていない。**

確認できた事実:
- NEC・ローソンいずれからも、2025年10月実証の「成果レポート」的なプレスリリースは発行されていない
- 作業分類精度、LLMレポートの品質、従業員フィードバック等の定量データは未公開
- 「従来の手計測データとの比較検証」を行ったことは報道されているが、その結果は不明

**考察**: パイロット期間がわずか1週間であったこと、結果をもとに技術改良を進めている段階と推測される。成果報告よりも技術のブラッシュアップを優先している可能性が高い。

### 3. 2026年以降の展開動向

#### 3-1. 追加店舗での実証 → 未確認

2026年4月時点で、ローソンの追加店舗での映像AI実証に関する公式発表は確認できなかった。

#### 3-2. 映像AI技術の進化（2026年3月19日プレスリリース）

NECは2026年3月19日、映像認識AI x LLMの次世代技術を発表:

- **「明文化されていない危険の予兆をAIで捉え、改善アドバイスを自動生成する技術」を世界初開発**
- 視覚言語モデル（VLM）を活用し、事前の作業定義や専門知識の体系化が不要
- 「指示されたものを探す」から「指示されていないものを見つける」への技術転換
- 物流・運送、製造業の現場映像分析に適用
- **2026年度中の実用化を目指す**
- 学術的裏付け: AAAI 2026、WACV 2026で論文発表済み

#### 3-3. cotomi Act（関連AIエージェント技術）

- NEC独自のAIエージェント技術「cotomi Act」が2025年12月にソリューション提供開始
- WebArenaベンチマークで人間超えの成功率80.4%を達成（世界初）
- 暗黙知をデータ化して学習する技術 → 映像AI技術と並行して2026年度に本格化予定

**考察**: ローソン実証の映像AI技術と、危険予兆検知技術・cotomi Actは同じ技術系譜上にある。ローソン実証は「小売店舗向け」、危険予兆検知は「物流・製造向け」と適用領域を分けて商用化を進めている構図。

### 4. 商用フェーズへの移行状況

**明確な商用化発表はない。** ただし以下の状況から、2026〜2027年度に商用移行する可能性が高い:

- 映像認識AI側: 2026年度中の実用化を公式に掲げている
- cotomi Actとの統合: 双方の商用化タイミングがほぼ同時期
- BluStellar戦略との整合: NECの次期中計（2026年度〜）で「BluStellar事業収益1兆円」が目標。映像AI系ソリューションはBluStellarの主要オファリングの一つ

### 5. NEC側のプロジェクト体制

#### 5-1. 組織構造（推定）

NECの公式組織体制から、ローソン案件に関わる可能性が高い組織:

| 組織名 | 役割推定 |
|--------|---------|
| **リテールソリューション事業部門**（エンタープライズBL配下） | ローソン向けアカウント管理、ソリューション提案の主管 |
| **流通・サービスソリューション事業部門** | コンビニ含む流通業向けソリューション全般 |
| **研究開発本部**（映像認識AI・VLM研究） | 映像AI技術の基盤開発、論文発表 |
| **BluStellar推進本部** | DX案件のシナリオ策定・バリュー提案 |

**注意**: 具体的な担当部門名・プロジェクトリーダーは公式に公開されていない。入社後に社内で確認が必要。

#### 5-2. NEC 2026年4月の組織改編

- 2026年4月1日、BluStellarをNECグループ全体（NESICホールディングス傘下含む）に展開
- DXの構想策定〜実装〜運用保守までEnd to Endで提供する体制を確立
- リテール向けもBluStellarシナリオの一部として位置づけ

### 6. 映像AI以外のNEC x ローソン プロジェクト

| **NEC顔認証決済** | 展開中 | 世界No.1精度の顔認証。ローソン直接導入の公式情報はないが、技術基盤として連携可能性あり |
| **NEC どこでもPOS** | 2025年〜 | スマホベースのモバイルPOS。コンビニ向け展開の可能性 |

### 7. KDDI x ローソンとの競合・補完関係

重要な発見として、**KDDIもローソンの映像AI・ロボティクス領域で大規模な実証を進めている**:

- **AIグラス実証**（2025年10月〜12月）: 従業員がAIグラス着用し業務撮影、AI分析・可視化
- **AI x ロボット実証**（2025年11月〜）: VLAモデルによる自律ロボットで品出し自動化
- **Real x Tech LAWSON** 1号店（2025年6月、高輪ゲートウェイシティ）: AIサイネージ、調理ロボ、清掃ロボ、飲料陳列ロボ

**NEC vs KDDI の位置づけ**:
- NEC: 映像認識AI（バックエンド分析）、セルフレジ/POS、データ分析（dotData）
- KDDI: AIグラス（フロントエンド支援）、ロボティクス、通信基盤、顧客データ連携

**両者は競合というよりも補完関係の可能性がある。** ただし「店舗作業30%削減」という同じ目標に向けて異なるアプローチを取っており、ローソン側がどちらの技術を本格採用するかは注視が必要。

### 8. KDDI x NEC のセキュリティ合弁

別角度の重要情報として:
- 2025年5月: KDDIとNECがサイバーセキュリティ事業で基本合意
- 2025年11月: 合弁会社「United Cyber Force（UCF）」設立、11月28日活動開始
- 能動的サイバー防御（ACD）領域で政府・法人向けソリューションを提供

**示唆**: KDDI-NEC間にはセキュリティ領域で強固なパートナーシップがある。ローソン案件においても、両社が対立するよりも協業する可能性がある。

## PMとしての示唆

### 入社直後のアクション

1. **映像AI実証の内部成果を確認する**: 公開されていないパイロットの定量結果（分類精度、LLMレポート品質、コスト対効果）を社内で入手し、次フェーズの提案に活かす
2. **リテールソリューション事業部門の組織図を把握する**: ローソンアカウント担当チーム、映像AI技術チーム、BluStellarシナリオチームの関係性を理解する
3. **KDDI連携の社内温度感を確認する**: UCF合弁の流れでローソン案件でもKDDI協業の議論があるか確認

### 提案機会

1. **映像AI x dotDataの統合提案**: 映像AIで取得した作業データとdotDataの購買データを組み合わせ、「この時間帯にこの作業を優先すべき」というレコメンドを出すシナリオ
2. **2026年度実用化フェーズのPM**: 危険予兆検知技術の物流・製造向け実用化がFY2026目標。リテール向けも同タイミングで動く可能性があり、PMとしてアサインを狙える
3. **ローソン x NEC x KDDI の三者協業PM**: セキュリティ合弁の関係性を活用し、映像AI（NEC）+ 通信基盤（KDDI）の統合ソリューションをPM視点で推進

## 未解明の論点

1. **パイロット実証の定量結果**: 作業分類精度、従業員の受容性、コスト試算 → 社内で確認必要
2. **追加実証の計画**: 埼玉1店舗以降の拡大計画があるのか → 社内ロードマップを確認
3. **ローソン側のベンダー選定基準**: NEC映像AI vs KDDI AIグラス、どちらを主軸にするかの意思決定プロセス
4. **Real x Tech LAWSONにおけるNEC技術の有無**: 高輪1号店のプレスリリースにNEC技術の言及がない。NEC排除なのか、別レイヤーでの関与なのか
5. **映像AIの商用価格体系**: SaaS型かプロジェクト型か、収益モデルの設計状況
6. **リテールソリューション事業部門の2026年度組織体制**: 4月の組織改編後の最新体制
