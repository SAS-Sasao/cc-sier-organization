# コンビニ業界用語集

NEC 入社・小売（コンビニ＝ローソン系）案件参画に向けたドメイン用語集。
業界構造 → 商品(MD) → 発注・物流 → POS・システム → 店舗運営 → 決済・顧客 → DX の順で体系化。
コンビニ特有の意味で記載しているため、一般小売と異なるニュアンスに注意。

- 作成日: 2026-05-31
- 対象: 小売ドメイン（コンビニエンスストア / ローソン系）
- 用途: 入社前ドメイン知識の習得・案件参画時のリファレンス

---

## A. 業界構造・フランチャイズ

| 用語 | 読み / 英 | 意味（コンビニ文脈） |
|---|---|---|
| FC | フランチャイズチェーン | 本部が商標・ノウハウ・システムを提供し、加盟店が出資・運営する仕組み。コンビニ大手の店舗の大半は FC 店 |
| 本部 | フランチャイザー | チェーン全体を統括。商品開発(MD)・システム・物流・販促を担う。ローソン本部など |
| 加盟店 | フランチャイジー / オーナー | 実際に店舗を運営する事業者。システムの「現場ユーザー」であり負担への配慮が要件に直結 |
| チャージ | ロイヤリティ | 加盟店が本部に支払う対価。コンビニでは売上ではなく「売上総利益(荒利)」に料率を掛ける方式が主流 |
| コンビニ会計 | 売上総利益分配方式 | 廃棄・棚卸ロスを売上原価に含めない独特の会計。**廃棄が増えても本部チャージが目減りしにくい**構造が、加盟店の廃棄抑制インセンティブと利害対立を生む論点 |
| SV | スーパーバイザー / 経営相談員 | 本部と店舗をつなぐ巡回担当。発注助言・売場づくり・販促指導を行う。SV 向けデータ提供もシステム要件になりうる |
| ドミナント戦略 | dominant | 特定エリアに集中出店し、物流効率・認知度・ブランド優位を高める出店戦略 |
| 直営店 | — | 本部が直接運営する店舗。新施策の実証(PoC)に使われやすい |
| チェーンストア理論 | — | 標準化・本部集中仕入による多店舗運営の経営理論。コンビニ運営の土台 |
| 24時間365日 | — | 原則無休・24時間営業。店舗システムは「止められない」前提（高可用性・オフライン会計が必須） |

## B. マーチャンダイジング(MD)・商品

| 用語 | 読み / 英 | 意味（コンビニ文脈） |
|---|---|---|
| MD | マーチャンダイジング | 「何を・いくつ・いつ・いくらで」売るかの商品政策全般 |
| 単品管理 | たんぴんかんり | コンビニ経営の原点。商品を1点(SKU)単位で売れ行き把握し、売れ筋投入・死に筋排除を回す |
| SKU | Stock Keeping Unit | 在庫管理の最小単位。同一商品でも容量・味違いは別 SKU |
| 売れ筋 / 死に筋 | うれすじ / しにすじ | 売れる商品 / 売れない商品。死に筋をカットし限られた棚を最適化 |
| フェイス / フェイシング | facing | 棚で商品の正面が見える数・配置。フェイス数は販売量に直結 |
| 棚割 | たなわり / プラノグラム(planogram) | どの棚のどこに何を何フェイス置くかの設計図。本部が標準を提供 |
| 定番 | ていばん | 常時取り扱う基本商品。対して「スポット/新商品」は入れ替わる |
| 日配品 | にっぱいひん | 弁当・おにぎり・調理パン・乳製品など毎日納品される鮮度命の商品。欠品・廃棄管理の最重要カテゴリ |
| 中食 | なかしょく | 弁当・惣菜など「買って持ち帰り家で食べる」需要。コンビニの主力かつ差別化領域 |
| FF | ファストフード(中食) | レジ横のホットスナック・揚げ物等(からあげクン 等)。粗利が高く重点商品 |
| PB / NB | プライベート/ナショナルブランド | PB=チェーン独自商品(高粗利・差別化) / NB=メーカー既製品 |
| カテゴリーマネジメント | — | 商品を「カテゴリ」単位で管理し、品揃え・棚・販促を最適化する手法 |
| 値入 / 荒利 | ねいれ / あらり(粗利) | 値入=売価と原価の差を設定すること / 荒利=売上総利益。コンビニの利益指標の中心 |
| 鮮度管理 | — | 賞味/消費期限の管理。日配品では時間帯単位の販売・廃棄判断が必要 |

## C. 発注・在庫・物流(SCM)

| 用語 | 読み / 英 | 意味（コンビニ文脈） |
|---|---|---|
| 発注 | はっちゅう | 店舗の最大の業務負荷かつ利益の源泉。欠品(機会損失)と廃棄(食品ロス)のトレードオフを握る |
| 推奨発注 | — | システムが過去実績・天候等から発注数を提案する仕組み |
| セミオート発注 | semi-auto | 推奨数を提示し人が確認・修正して確定する半自動発注（ローソンが導入） |
| AI.CO | アイコ | ローソンの AI 発注システム。発注業務時間を約4割削減を標榜、2024年から順次展開 |
| 自動発注 / AI発注 | — | AI が需要予測に基づき発注数を自動算出。人手不足対策の本丸 |
| 欠品 | けっぴん / out of stock | 在庫切れによる販売機会損失(機会ロス)。顧客離反にも直結 |
| 廃棄ロス | 食品ロス / waste | 売れ残り・期限切れによる廃棄。1店舗あたり1日 約10〜15kg とされ社会課題化 |
| 賞味期限 / 消費期限 | — | 賞味=おいしさの目安 / 消費=安全に食べられる期限(弁当等)。発注・値引き判断の前提 |
| 多頻度小口配送 | — | 少量を1日複数回届ける配送。鮮度維持と省スペースを両立するコンビニ物流の特徴 |
| 定温物流 | — | 温度帯別(米飯/チルド/フローズン/ドライ)に分けた配送。混載できず便を分ける |
| 共同配送 / DC | 共配センター / Distribution Center | メーカー横断で取りまとめ効率配送する専用センター |
| リードタイム | lead time | 発注から納品までの時間。短いほど精度の高い発注が可能 |
| 見切り | みきり | 期限が近い商品の値引き販売。廃棄削減策(エシカル消費)。ポイント還元で促す例も |
| ABC分析 | — | 売上貢献度で商品を A/B/C にランク分けし管理に強弱をつける手法 |

## D. POS・システム・データ

| 用語 | 読み / 英 | 意味（コンビニ文脈） |
|---|---|---|
| POS | Point of Sales | 販売時点情報管理。レジでの販売データを単品単位で収集する仕組み。単品管理の基盤 |
| POSデータ | — | いつ・何が・いくつ売れたかのデータ。発注・棚割・MD の意思決定に使う一次データ |
| 客層キー | — | コンビニ POS 特有。レジ打ち時に客の推定年齢・性別を入力するボタン。客層別分析の元データ |
| ストアコントローラ | storcon / SC | 店舗内システムの中核サーバ。POS・発注端末・在庫・本部通信を統括（コンビニ基幹の要） |
| 基幹システム | — | 発注・在庫・会計・販売など店舗運営の根幹を担うシステム。刷新は省人化の鍵（業界で端末刷新が進行中） |
| 情報系システム | — | POS データ分析・需要予測・販促など意思決定支援を担う系統 |
| 商品マスタ | — | 全商品の名称・価格・カテゴリ・JAN を管理する基礎データ。整合性が全機能の前提 |
| JANコード | バーコード | 商品識別の標準バーコード。レジスキャン・在庫の基礎 |
| ID-POS | — | 会員ID(ポイント等)と購買を紐づけたデータ。誰が買ったかが分かり One to One 施策に活用 |
| EOB / 発注端末 | Electronic Order Book | 売場で発注するハンディ/タブレット端末。バックヤード大型端末からの置き換えが進行 |
| セルフレジ / セミセルフ | self / semi-self | 客が会計操作 / 登録は店員・支払いは客、の分担型。省人化施策 |
| バックヤード | back yard | 店舗の事務・在庫スペース。発注・検品作業の場。端末の小型化・店頭化が進む |

## E. 店舗運営・オペレーション

| 用語 | 読み / 英 | 意味（コンビニ文脈） |
|---|---|---|
| 店舗オペレーション | — | 発注・品出し・接客・清掃・レジ等の店舗業務全般。ローソンは2030年度に作業の3割削減を目標 |
| 品出し / 前出し | フェイスアップ | 在庫を棚に補充 / 商品を手前に揃えて見栄えと欠品感を解消する作業 |
| 棚卸 | たなおろし | 実在庫を数えて帳簿と突合する作業。ロス把握の基礎 |
| 波動 | はどう | 時間帯・曜日・天候による来店/売上の変動。朝・昼ピーク等。シフト・発注設計に影響 |
| ピークタイム | — | 来店集中時間帯(出勤前・昼・夕方)。レジ待ち・欠品が起きやすい |
| ワンオペ | — | 1人勤務体制。人手不足の象徴的問題。省人化技術の動機 |
| QSC | Quality / Service / Cleanliness | 品質・サービス・清潔さ。店舗評価の基本軸 |
| クレンリネス | cleanliness | 清潔さの維持。来店動機・ブランドに直結 |
| シフト / レイバースケジューリング | — | 人員配置計画。波動に合わせた最適化が省力化テーマ |

| デジタルツイン | digital twin | 店舗・サプライチェーンを仮想空間に再現しシミュレーション最適化 |
