```
エンタープライズBU
 ├── 金融ソリューション事業部門
 ├── 製造ソリューション事業部門
 ├── リテールソリューション事業部門  ← ここ

 │    │    ├── 第三インテグレーショングループ（大手コンビニのDX・システムソリューション）

 │    └── ...

```

**結論: リテールは「クロスインダストリーBU」ではなく「エンタープライズBU」配下。**
2025年4月のクロスインダストリーBU → パブリックBU集約は行政・医療が対象であり、リテールは影響を受けていない。

### ローソン業務の位置づけ

- 第一リテールソリューション統括部が大手コンビニ（セブン・ローソン等）を担当
- NECソリューションイノベータ（NESIC）にも「EA・リテールソリューション事業部門」がグループ会社として並走
- セブン次世代システムでは NEC + NECソリューションイノベータ が共同で開発を実施

### 2025年4月の組織再編で知っておくべきこと

1. **クロスインダストリーBUの行政・医療事業 → パブリックBUに集約**（リテールは無関係）
2. **デジタルプラットフォームBUが2分割**: サービスBU（BluStellar製品）+ デリバリーBU（SI/コンサル）
3. **テレコムBUは整理縮小方向**: 採算性の低い事業を整理。利益率10%以上を目標
4. **ANSに海底ケーブル事業を移管**: テレコムBUから移管

## 3. 利益率目標がPMにとって何を意味するか

### マクロ: 全社利益率目標の構造

```
現在（2024年度）          目標（2030年度頃）
全社 Non-GAAP OP率 7.5%  → 15.0%（2倍）
BluStellar OP率   12.2%  → 20.0%（+8pt）
```

### 利益率向上のメカニズム（IR資料から読み取れる4つの柱）

#### 柱1: シナリオ型ビジネスへのシフト（最重要）

| 分類 | 利益率 | 特徴 |
|------|--------|------|
| BluStellar シナリオ | 最も高い（ベースより3〜5%高い） | 顧客課題に合わせた包括提案。コンサル→デリバリー→運用の連鎖型 |
| BluStellar オファリング | 中間 | NEC製品・サービスの組み合わせ |
| BluStellar プロダクト&サービス | ベースライン | ハード・ソフト単品 |
| ベース事業（従来SI） | 最も低い | 人月積み上げ型 |

**PMへの翻訳**:
- シナリオ型 = 「ローソンの経営課題を起点に、NECの複数ソリューションを組み合わせて提案 → コンサルから運用まで一気通貫で受注 → 長期継続契約」
- 従来SI型 = 「ローソンから『このシステム作って』と言われて人月で見積もる → 単発受注 → 低マージン」
- **BluStellar 20%利益率の正体は「シナリオの構成比を33%→63%に引き上げること」で達成するもの**

#### 柱2: バリューベースプライシング

- 従来: 人月 × 単価 = 見積もり（工数ベース）
- 目標: 「成果に支払う」仕組み
  - 3ヶ月単位での成果に対する費用
  - 成果が出た金額に対するパーセンテージで対価
- PMとしての行動変容: 「何人で何ヶ月」ではなく「何を達成したらいくら」で提案書を書く

#### 柱3: 再利用・横展開による効率化

- シナリオを「モデル化して横展開」→ 似た業種・課題に低コストで再提案
- PMとしての行動変容: ローソンで成功した施策を他のコンビニ・小売に展開する視点を常に持つ

#### 柱4: AI活用による生産性向上

- 経営管理システムでのAI実装でリードタイム96.7%削減の実績
- 「cotomi」等のLLM活用で設計・コーディング・テスト工数を削減
- PMとしての行動変容: AI駆動開発スキルが直接的に利益率改善に貢献

### プロジェクトレベルでの利益率目標（推定）

NECはプロジェクト単位の利益率目標を公開していないが、以下から推定できる:

- BluStellar全体で20%営業利益率を目標
- 営業利益率 = 売上 - 原価 - 販管費
- SIerの粗利率は一般的に30〜40%（原価の大半は人件費+外注費）
- 営業利益率20%達成には、粗利率35%程度 + 販管費率15%程度が必要

**→ プロジェクト粗利率で30〜40%が期待値と推定。**
**→ 従来SI案件の粗利率が25%前後だとすると、シナリオ型で35%以上を狙う必要がある。**

### 「価値創造型」vs「SI型」 — 財務的な違いの整理

| 項目 | 従来SI型 | 価値創造（BluStellar シナリオ）型 |
|------|---------|-------------------------------|
| 単価設定 | 人月×単価（工数ベース） | バリューベース（成果ベース） |
| 契約形態 | 請負・準委任（単発） | コンサル→SI→運用の連鎖（長期） |
| 利益率 | 低い（ベースライン） | 高い（+3〜5%以上） |
| 収益の予測可能性 | 案件ごとにリセット | 長期契約でストック型 |
| 提案の起点 | 顧客の要件書 | 顧客の経営課題 |
| NEC側の関与範囲 | 設計〜開発〜テスト | 構想策定〜実装〜運用・保守 |
| PMに求められるスキル | PM力（QCD管理） | PM力 + コンサル力（課題設定能力） |

### 評価の2軸（9ブロックマトリクス）

**横軸: 業績評価**（期初に立てた業績目標の達成度と成果）
**縦軸: 行動評価**（5つのCode of Valuesの実践度）

各軸を「期待を上回る / 期待どおり / 期待を下回る」の3段階で分類 → 3x3 = 9ブロックに配置

### 5つのCode of Values（行動評価の基準）

1. **視線は外向き、未来を見通すように** — 顧客・市場を見ているか
2. **思考はシンプル、戦略を示せるように** — 複雑なものを整理して方向性を示せるか
3. **心は情熱的、自らやり遂げるように** — 主体的にやり切るか
4. **行動はスピード、チャンスを逃さぬように** — 素早く動くか
5. **組織はオープン、全員が成長できるように** — チーム・組織に貢献するか

### 中計目標と評価の接続（PM視点での読み替え）

| 中計目標 | PMの業績評価で測定されるもの（推定） |

| BluStellar売上1兆円 | 担当案件のBluStellarへの貢献度（シナリオ/オファリング比率） |
| 営業利益率20% | **プロジェクト粗利率**（予算対実績） |
| シナリオ構成比63% | 従来SI型→価値提案型への転換実績 |
| 全社Non-GAAP OP率15% | 高収益案件の獲得・完遂 |

### PMが評価されるポイント（推定・IR資料+求人情報から）

1. **プロジェクトのQCD達成**: 期限通り、予算内（粗利率目標達成）、品質基準クリア
2. **売上貢献**: 担当案件の受注金額・売上金額
3. **利益率**: プロジェクト粗利率の管理。BluStellarシナリオ型の方が評価が高い
4. **顧客満足度**: リピート受注、追加受注の有無
5. **Code of Values実践**: 5つの行動基準の体現度

### 株式報酬制度「NECバリュー・シェアズ」

- 将来的にグループ6,000人以上に拡大予定

## 6. まとめ: PMの日常業務レベルへの翻訳

### 数字の階層構造

```
全社 Non-GAAP OP率 15%（2030年目標）
  ↓
ITサービス OP率 13%前後（現在）→ 上昇方向
  ↓
BluStellar OP率 20%（次期中計目標）
  ↓
シナリオ型案件の利益率 = ベース+3〜5%
  ↓
プロジェクト粗利率 30〜40%（推定目標）
  ↓
PM: 「人月積み上げ」ではなく「価値ベースの提案」で案件を取る
```

### PMとして毎日意識すべき3つのこと

1. **プロジェクト粗利率を常にウォッチする**: BluStellar 20%という会社目標は、プロジェクトレベルで30〜40%の粗利率として降りてくる。工数管理が甘いと即座に利益率を食う
2. **シナリオ型の提案ができるか**: 「ローソンから言われたものを作る」だけでは従来SI型。ローソンの経営課題（日販70万円、KDDI連携、店舗省人化）を起点に、NEC側から提案する力がBluStellar戦略のど真ん中
3. **Code of Valuesを体現する**: 特に「視線は外向き」と「行動はスピード」。ローソンのビジネスを理解し、素早く動くPMが評価される
