# 小売ドメイン室

## 役割
日本の小売業界のドメイン知識収集・整理を担当。業界用語、業務プロセス、事例の体系化を行う。

## 対象領域
- 小売業の業務プロセス（仕入、在庫管理、販売、EC）
- 業界用語・略語の整理（POS、MD、SKU、棚割、フェイス等）
- 業界動向・事例（オムニチャネル、DX、データ活用）
- システム構成パターン（基幹系、情報系、EC系）

## ファイル操作ルール
- 業界レポートは `industry-reports/{topic-id}.md` に配置
- 用語集は `glossary/` に配置
- 事例は `case-studies/{case-id}.md` に配置
