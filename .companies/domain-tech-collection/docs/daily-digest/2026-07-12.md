## C. クロスドメイン分析

### 1. ループエンジニアリングとAIエージェント活用が小売DXを加速

AIエージェントをループ稼働させる「ループエンジニアリング」の手法が技術コミュニティで急速に普及している。ハーネス設計やサンドボックス構築の知見が成熟する中、小売側ではローソンのアバター接客実証やEBISU AGENTによるEC運用自動化が始まっている。SIerとしては、これらのエージェント基盤技術を店舗オペレーション改善やEC運用の自動化に適用する提案が差別化ポイントになる。BunのZig→Rust書き換えに見られる64並列Claude Codeの活用事例は、大規模レガシーシステム移行案件にも応用可能な知見を含んでいる。

### 2. LLMモデル世代交代と小売業の業務AI活用余地

Fable 5、Opus 4.8、GPT-5.6 Sol、Grok 4.5と主要AIモデルが相次いでリリース・比較され、性能が急速に向上している。一方で「AIデスキリング」への警鐘も鳴らされている。小売業では、ワークマンの580円Tシャツ80万枚売上のようなヒット商品開発力がある一方、AIによる需要予測やMD自動化の導入余地は大きい。SIerとしては、最新LLMのコスト性能比を踏まえた小売向けAIソリューション（需要予測・自動棚割・チャットbot接客）の提案が有効である。

### 3. AWS基盤のセキュリティ強化とEC決済安全対策の連動

AWS MCP ServerのOAuth 2.1対応やAWS Config 191ルール追加など、クラウド基盤のセキュリティが強化されている。小売EC側では楽天・ヤフー・メルカリが警察庁と安全確保協定を締結し、EMV 3-Dセキュア普及で不正利用被害が減少傾向に入った。一方で決済代行「全東信」の破綻は、決済インフラの脆弱性を改めて浮き彫りにしている。SIerとしては、クラウドセキュリティとEC決済セキュリティを一体的に設計する統合提案が差別化につながる。

### 4. 主要小売の好決算とデジタル投資余力の拡大

イオン営業利益33.6%増、セブン＆アイ61.4%増、良品計画2桁増収増益と、主要小売企業の四半期決算が軒並み好調である。一方でコンビニ6月既存店売上は天候不順で苦戦し、来店頻度依存型ビジネスのリスクが顕在化した。好決算企業はDX投資余力が拡大しており、イオンモールのベトナム初出店やローソンのアバター接客など、テクノロジー投資案件が増加する見込み。ANAのシステムリニューアル障害は、大規模更改時のリスク管理の重要性をSIerに改めて示している。

### 5. 物流DXの進展とラストワンマイル改善

トラック拘束時間が前年比1時間33分短縮、宅配再配達率7.6%と物流改善が数値に表れている。Uber Japanの即時配送が47都道府県に拡大し、越境EC返品のAI効率化も進む。一方で物流現場の6割が人手不足を実感しており、自動化投資は待ったなしの状況にある。SIerとしては、TikTok Shop経由の消費額46%増に伴う物流需要増への対応として、EC企業向けの物流最適化やWMS刷新提案が有望な領域である。
